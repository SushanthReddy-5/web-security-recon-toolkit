# Web Security Recon Toolkit

A Python-based Web Reconnaissance and Security Assessment Toolkit developed for learning practical web penetration testing and reconnaissance concepts.

## Features

- HTTP response analysis
- Security header detection
- Security posture scoring
- Endpoint enumeration
- Common port scanning
- Basic banner grabbing
- Technology fingerprinting

## Security Checks Performed

- X-Frame-Options
- Content-Security-Policy (CSP)
- HTTP Strict Transport Security (HSTS)

## Ports Scanned

- 21 (FTP)
- 22 (SSH)
- 80 (HTTP)
- 443 (HTTPS)
- 3306 (MySQL)

## Technologies Used

- Python
- requests module
- socket programming

## Purpose

This project was developed to improve practical understanding of:

- Web reconnaissance
- Attack surface analysis
- HTTP/HTTPS communication
- Security headers
- Service enumeration
- Basic penetration testing methodology
